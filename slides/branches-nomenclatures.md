##NOMENCLATURES

- **TITRE**  
une utilité, une spécificité, …  
*exemples* : « feature », « fix », « mobile », « IE »

- **DESCRIPTION**  
résumé de son utilité  
*exemples* : « navigation », « smooth-scroll », « traductions »

- **LANGUE**  
français ? anglais ?
- éviter les mélanges de langues
- c’est pas interdit (feature, bug, swipe, …)
